# Developing using docker

## Installation of docker

Make sure to have an installation of Windows 10 Education or use mac (Free licence through UCL) 

Download docker for windows or docker for mac

## Cloning the repository

Install uclapi into a common source folder on your computer using:

git clone https://github.com/uclapi/uclapi.git

## Building the frontend

cd frontend

npm start
